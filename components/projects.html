Data4Good 2025 - Berlin

This project uses data of the Holocaust prisonners and it aims to show and illustrate their histories. It presents an interactive map of the concentration camps, individual prisonners routes during that time and their stories.

Image Captioning

We extended an open-source CNN-RNN image captioning model from GitHub by adding visual attention for interpretability and top-p sampling for diverse outputs. We also developed a UI to test captions and visualize attention maps.

Start HACK 2025 - St Gallen

We developed a financial advisor to help wealth managers analyze their clients' portfolios and behavior, explore market trends and make AI-based recommendations.
